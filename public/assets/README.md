# Assets

Drop brand assets here. Files in `public/` are served at the site root, so a
file saved as `public/assets/goldsure-logo.png` is available at
`/assets/goldsure-logo.png`.

## Logo

Add your Goldsure logo as:

- `goldsure-logo.png` — shown in the app header and on the printed/emailed
  pickup slip (Stock Release Request).

Recommended: a horizontal PNG with a transparent background, at least 320px
wide. If the file is missing the app simply hides the logo, so nothing breaks
before you add it.
